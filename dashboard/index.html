📊 KEY TAKEAWAYS FROM PREDATOR REPORT

Top 5 Lessons for New Builders:

1️⃣ EOA_DIRECT from day one — Executor contracts cost us ~14.78 XRP in trapped payouts. For simple buy/claim markets, skip the executor layer entirely.

2️⃣ 5-Bucket Accounting Model — Wins, Clean Losses, Executor Losses, Refunds, Voids. Naive win/loss counting was wrong by 10.08 XRP until we fixed this. Refunds and voids are NOT real decisions.

3️⃣ Long-running scripts > Cron — Cron jobs cost RLUSD per execution (~5/day at 5-min intervals). Use supervisor.sh + background Python scripts instead. Zero token cost.

4️⃣ KV-backed public dashboard — No tunnel dependency. Survives bot restarts. Cloudflare KV + Pages Functions = robust, decoupled architecture.

5️⃣ Contested market filter — Only bet where opposing liquidity exists. Sole-bettor bets have zero ML training value and are often refunds. Our contested WR (56%) is the true signal quality metric.

Biggest Regrets:
- Over-engineering with executor contracts
- Not implementing accounting buckets early
- Tunnel-dependent first dashboard design
